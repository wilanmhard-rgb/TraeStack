## 中文

TraeStack 是一个面向 Trae 的 skill 定义仓库，提供可发现、可路由、可稳定调用的技能目录与说明文件。

- 快速开始：`python3 scripts/validate_skills.py`
- Linux 部署：`bash scripts/deploy-linux.sh`
- macOS 部署：`bash scripts/deploy-macos.sh`
- Windows 部署：`powershell -ExecutionPolicy Bypass -File .\scripts\deploy-windows.ps1`
- GitHub 发布说明：`GITHUB-PUBLISH.zh-CN.md`

完整中文文档见 [README.zh-CN.md](./README.zh-CN.md)。
